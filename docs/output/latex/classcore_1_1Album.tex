\doxysubsection{Descrição detalhada}
Representa um álbum musical com suas músicas. 

A classe \doxylink{classcore_1_1Album}{Album} armazena informações como nome, artista, gênero e a lista de músicas contidas no álbum. Mantém a ordem original das faixas para reprodução sequencial.
